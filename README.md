# Password Strength Analyzer

## Overview
Password Strength Analyzer is a Python-based tool that evaluates the strength of user-entered passwords.

## Features
- Checks password length
- Verifies uppercase letters
- Verifies lowercase letters
- Verifies numbers
- Verifies special characters
- Classifies passwords as Weak, Medium, or Strong
- Provides suggestions to improve password strength

## Technologies Used
- Python 3
- Regular Expressions (re module)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/amuthanabinesh-creator/Password-Strength-Analyzer.git
```

2. Open the project folder.

3. Run:

```bash
python password_analyzer.py
```
